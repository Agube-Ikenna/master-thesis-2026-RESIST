# Master Thesis 2026 RESIST

This repository contains the working materials, datasets, scripts, outputs, and writing files for my MSc thesis in Environmental Toxicology.

## Repository structure

### config/
Project-level configuration files.

### data/
- raw/ → Original source data
- processed/ → Cleaned and analysis-ready datasets
- metadata/ → Data dictionaries and study design info

### docs/
- admin/ → Thesis forms, guidelines
- literature/ → Organized references
- notes/ → Working notes and guides
- papers/ → PDFs of articles and theses
- protocols/ → Methods and device manuals

### reports/
- drafts/ → Thesis drafts and working documents
- presentation/ → Slides
- thesis/ → Final thesis outputs

### results/
- figures/ → Plots and images
- tables/ → Final tables
- model_outputs/ → Saved models and R outputs

### scripts/
- 00_setup/ → Setup scripts
- 01_cleaning/ → Data cleaning
- 02_processing/ → Data transformation
- 03_analysis/ → Main analysis
- 04_models/ → Model scripts
- 05_visualization/ → Plot scripts

### archive/
Legacy projects, duplicates, and old versions (read-only)

## Working rules

- Raw data → data/raw/
- Clean data → data/processed/
- Scripts → scripts/
- Outputs → results/
- Writing → reports/
- Literature → docs/papers/
- Old stuff → archive/
